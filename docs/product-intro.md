# 鲛人书 Arielgram

> 口述成图，如鲛人织绡 —— 一句自然语言，即生成一张专业示意图。

## 一句话

**鲛人书** 是一个 AI 驱动的图表生成助手。用户在飞书中用自然语言描述需求，Bot 即时回复带有预览图的 Mermaid 代码，可一键粘贴到飞书文档渲染。

## 名字由来

**鲛人**，《搜神记》中记载的南海水族，"水居如鱼，不废缉绩"，其泪成珠、其织成绡。**鲛人书** 寓意如同鲛人织绡一般，将源源不断的自然语言"丝线"编织成精密优雅的图表。

英文名 **Arielgram** = Ariel（莎士比亚《暴风雨》中的空气精灵 + 小美人鱼） + -gram（图表），与中文名呼应。

## 核心功能

### 自然语言 → 图表 + 预览

```
用户: 画一个 HWP 状态机，包含 OFF、Standby、Active、Override、Failure
Bot:  [状态机预览图] + Mermaid 代码
```

支持图表类型：**流程图、时序图、状态图、类图、ER 图、甘特图**。

### 对话式迭代

Bot 按用户维护独立上下文。后续指令自动在前一张图基础上修改，无需重复描述整体结构。

```
用户: 把 FAILURE 改成 FAULT，其他不要变
Bot:  [局部修改后的新图]   ← 其他部分与原图一致
```

用户说 `/reset` 即清空上下文，开启全新任务。

### 交互式卡片体验

- 先回一张"思考中"蓝色卡片
- 生成完成后**原地更新**为预览图 + Mermaid 代码
- 一张卡片内完成从提问到结果的全流程

### 多用户隔离

每个用户的对话上下文独立维护，互不干扰。24 小时无活动自动过期。

## 目标用户

汽车智能驾驶领域的产品经理、Feature Owner、系统工程师、架构师。

内置智驾领域术语词典（HWP、NOD、ODD、ASIL、CAN/ETH/LIN 等），也适用于通用软件架构场景。

## 使用方式

在飞书群聊或单聊中 **@机器人** 发送指令：

```
@鲛人书 画一个传感器架构图，包含摄像头、毫米波雷达、激光雷达、域控制器
@鲛人书 把域控制器拆成感知MCU和规划MCU
@鲛人书 画一个变道决策的时序图
@鲛人书 /reset
```

也提供 Web UI（浏览器访问 `http://localhost:3001`），适合不依赖飞书的场景。

## 技术特点

| 特性 | 说明 |
|------|------|
| 零公网依赖 | 飞书 WebSocket 长连接，无需公网 IP 或域名 |
| 本地部署 | 代码 + API Key 即可运行，数据不出境 |
| 即时预览 | mermaid.ink 在线渲染为 PNG 嵌入卡片 |
| 跨平台 | Windows / macOS / Linux 均可 |
| 轻量 | 纯 Node.js，无数据库，内存 Session |
