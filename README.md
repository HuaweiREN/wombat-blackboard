# Wombat-Blackboard

自然语言 → Mermaid/PlantUML 代码 → 飞书画板一键出图

## 一句话

用自然语言描述场景，即可生成可粘贴到飞书画板的 Mermaid/PlantUML 代码。

## 目标用户

汽车智能驾驶系统的产品经理、Feature Owner、系统工程师、架构师。

## 项目结构

```
wombat-blackboard/
├── docs/
│   └── PRD.md            # 产品需求文档
├── prompts/               # Prompt 工程文件
├── templates/             # 领域图表模板
├── src/                   # 源码
└── tests/                 # 测试用例
```
